# 🎬 Movie Search App

A responsive web application that allows users to search for movies and view movie details dynamically using the OMDb REST API.

## 📂 GitHub Repository

🔗 https://github.com/karthik19-ch/Public-REST-API

## 📖 Project Overview

This project was developed as part of an internship task to demonstrate the use of a public REST API. The application fetches movie data from the OMDb API and displays results dynamically on the webpage using JavaScript Fetch API.

## ✨ Features

- Search movies by title
- Fetch data from a public REST API
- Dynamic content rendering
- Loading indicator while fetching data
- Error handling for invalid searches
- Responsive user interface
- Display movie posters
- Show movie release year and type

## 🛠️ Technologies Used

## ⚙️ How It Works

1. Enter a movie name in the search box.
2. Click the Search button or press Enter.
3. The application sends a request to the OMDb API using Fetch API.
4. A loading indicator is displayed while data is being fetched.
5. Search results are rendered dynamically on the page.
6. Error messages are shown if no results are found or if an issue occurs.
